# CampusConnect

A web-based campus social platform built with Firebase.

## Features

- User authentication (Login/Sign Up)
- Dashboard with posts
- User profiles

## Tech Stack

- HTML/CSS/JavaScript
- Firebase (Authentication, Firestore, Storage)

## Setup

1. Configure Firebase credentials in `js/firebase.js`
2. Open `public/index.html` in a browser or use a local server

## Project Structure

```
campusconnect/
├── public/          # HTML pages
├── js/             # JavaScript modules
├── css/            # Stylesheets
└── assets/         # Static assets
```
